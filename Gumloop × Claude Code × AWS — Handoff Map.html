Friction Points & Resolution Path
🔴 Now — Manual seam
The Gumloop → Claude Code handoff is manual. You paste the spec + context into Claude Code's terminal yourself.

Mitigation: Standardise a context template (spec + schema + constraints) so the paste is consistent every time. Takes ~2 min per task.
🟡 Soon — Semi-automate
Gumloop generates the packaged context file automatically and drops it in a shared Drive folder. You open it, paste into Claude Code, commit. One step instead of three.

Requires: A Gumloop "context packager" automation (buildable now).
🟢 Later — Full loop
When Claude Code exposes an API, Gumloop calls it directly. Spec change → code update → GitHub commit → AWS deploy, zero human steps.

Requires: Anthropic's Claude Code API (not yet public). ETA unknown.
